# Portfolio

Personal portfolio website for Sai Prasath S, AI & ML Engineer.

Built from scratch with a **Natural Organic** design archetype, featuring dual-mode custom styling (dark/light themes), smooth CSS animations, intersection observers, and lightweight interactions.

## Features Included
- **Natural Organic Theme:** Cohesive deep espresso and warm alabaster palettes with sage and ochre accents.
- **Reading Progress Bar:** Smooth scroll tracking indicating content depth.
- **Circular Theme Transition:** Unique circular wipe expansion when toggling light/dark modes.
- **Time-Based Hero Greeting:** Dynamic greets based on visitor local time (morning/afternoon/evening).
- **Interactive Technical Grids & Filters:** Staggered viewport entries and category tabs filtering skills dynamically.
- **Parallax Hero blobs:** Moving vector background shapes following desktop pointer coordinates.
- **Animated Stats Count-up:** Scroll-driven easing animations showing quantified experience achievements.
- **Full Responsiveness:** Optimized burger navigation menus, fluid clamp typography, and stacked grids matching mobile/tablet viewports.
- **SEO & Structure Optimized:** Full semantic HTML5 tags and schema JSON-LD indexing records.
- **Contact Form Submission:** Web3Forms integration handling API validations and status indications.
